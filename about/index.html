  • Reinforcement and Transfer Learning Basics

  1. Markov Decision Process (MDP): \(M =(\mu_0,S,A,T,\gamma,R)\)

\(\mu_0\), initial states, \(S\) is the current state. \(A\) is the action based on current state,

\(T\) is the transition probability distribution, namely \(T(s'|s,a)\), specifying the probability od the state transitioning to \(s'\) upon taking action \(a\) from state \(s\).

\(R\) is the reward distribution, namely \(R(s,a,s')\) is the award that an agent can get by taking action a from state s with the next state being s'.

  1. Policy function \(\pi(a|s)\), denotes the probability of taking action \(a\) from state \(s\).

  2. Value function \(V_M^\pi(s)=\mathrm{E}[\sum_{i=0}\gamma^ir_i;\pi;s]\), estimates the quality of being in state \(s\).

  3. Q-function, similar to the value function.

  4. RL goals: maximizing the expected return: \(J(\pi) = \mathrm{E}_{(s,a)~\mu^\pi (s_a)}[\sum_{i=0}\gamma^ir_i]\)

  5. Common algorithms: Monte-Carlo, Temporal Difference, Poliy Gradient

  6. Transfer Learning: aims to learn an optimal polict \(\pi^*\) for the target domain \(M_t\) by using information \(I_s\) from source domain \(M_s\):

\[ \pi^* = \mathrm{arg max}_{\pi}\mathrm{E}[Q_M^\pi(s,a)]\nonumber \]

where \(\pi\) is a policy learned for the target domain based on information from both \(I_t\) and \(I_s\)
